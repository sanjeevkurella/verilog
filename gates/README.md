# Verilog Logic Gates

This repository contains the implementation of basic logic gates in Verilog HDL.

## Gates Implemented

* AND Gate
* OR Gate
* NOT Gate
* NAND Gate
* NOR Gate
* XOR Gate
* XNOR Gate

## Modeling Styles

Each gate is implemented using:

1. Structural Modeling
2. Dataflow Modeling


## Files

Each gate directory contains:

* *_st.v : Structural Modeling
* *_data.v : Dataflow Modeling
* *_data_always.v : Behavioral Modeling
* *_tb.v : Testbench
* Waveform Screen
